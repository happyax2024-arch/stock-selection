启动方式

cd C:\Users\MI\stock-screener
python -m streamlit run app.py
打开浏览器访问 http://localhost:8501

关键特性
5个可选条件：左侧勾选，支持多选组合
股票数限制：滑块控制扫描范围（100-5500），默认500只，避免首次加载过慢
数据缓存：日线缓存4小时，实时行情60秒，首次加载后后续秒开
结果导出：支持CSV和Excel格式
数据源：默认使用Sina（更稳定），东财作为备选
注意：今天是5月1日劳动节，A股休市，数据为4月30日收盘数据。条件2（竞价数据）在非交易时段可能无结果。条件4的获利筹码计算依赖 fengwo 模块（仅Windows），当前环境已自动使用Python回退算法。
